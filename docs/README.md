# Vesy13 Documentation

Vesy13 - WinForms-приложение для вагонных весов N13. Основная задача системы:
считать показания АЦП "СИМ А04", выполнить статическое или динамическое
потележечное взвешивание, сохранить результат в локальную PostgreSQL-базу,
затем перенести или скорректировать запись в заводской Firebird-базе.

## Быстрый старт для разработчика

1. Установить .NET SDK 8 с поддержкой Windows Desktop.
2. Подготовить локальную PostgreSQL-базу `scale_db` и применить SQL из `migrations/`.
3. Проверить доступность Firebird-базы `DC_OPER.FDB`, если нужны перенос, корректировки и печать.
4. Проверить имя COM-порта АЦП в `settings.json` или через сервисную форму.
5. Собрать решение:

```bash
dotnet build Vesy13.sln
```

Основной проект: `Vesy13.csproj`.

Вспомогательный тестовый эмулятор COM-порта: `ScaleListener/ScaleListener.csproj`.

## Важные замечания

- Строки подключения к PostgreSQL, Firebird и Application Insights сейчас находятся в коде.
- `settings.json` создается рядом с исполняемым файлом при первом запуске.
- Настройки администратора хранятся в `settings.json` как PBKDF2-хеш и соль.
- Приложение рассчитано на Windows: WinForms, COM-порт, Firebird-путь Windows и P/Invoke для аудита пользователя.
